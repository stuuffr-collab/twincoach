## Defect Severity Template

### P0
- Broken critical learner path
- Data corruption
- Session progression corruption
- Submit loop unusable

### P1
- Misleading learner state
- Wrong redirect behavior
- Wrong feedback/result handling
- Major usability break in onboarding or diagnostic

### P2
- Non-blocking UI issue
- Minor copy issue
- Cosmetic inconsistency

## Defect Report Fields
- Title
- Severity
- Route
- Preconditions
- Steps to reproduce
- Expected result
- Actual result
- Contract affected
- Owner
- Status
